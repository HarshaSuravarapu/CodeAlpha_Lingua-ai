 🌍 LinguaAI — Smart AI Language Translator

LinguaAI is a modern AI-powered language translation web application built using HTML, CSS, and JavaScript.  
It provides real-time language translation with a futuristic UI, voice input, speech output, and responsive design.

---

# 🚀 Features

✨ Beautiful Modern UI  
🌈 Animated Gradient Background  
🧊 Glassmorphism Design  
🌐 Multi-language Translation  
🎤 Voice Input (Speech Recognition)  
🔊 Text-to-Speech Output  
📋 Copy Translated Text  
🔄 Swap Languages  
📱 Fully Responsive Design  
⚡ Fast Translation API  
🎨 Smooth Hover Animations  

---

# 🖥️ Demo Preview

## Main Features

- Enter text in any supported language
- Select source language
- Select target language
- Translate instantly
- Listen to translated speech
- Use microphone for voice input
- Copy translated text
